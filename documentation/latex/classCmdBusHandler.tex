\subsection{Detailed Description}
The cmd bus class. 

Description: This class models an address and command bus. The cmd issuer sends a cmd containing address to this. it takes that command and delivers it to the \doxyref{DRAM}{p.}{classDRAM} after t\_\-CMD cycles.
