\subsection{Detailed Description}
Class to model cmd, address and data buses of a \doxyref{DRAM}{p.}{classDRAM}. 

Description: This class just acts a wrapper around the data and cmd buses of all the channels. It contains the array of both cmd and data bus handlers. It is not used during normal \doxyref{MC}{p.}{classMC} \& \doxyref{DRAM}{p.}{classDRAM} operations but only used to set up some links between the busses and other components in the beginning
