\subsection{Detailed Description}
The data bus class. 

Description: This class handles the operation of the data bus. It receives read and write data from the corresponding dram channel or cmd issuer, respectively. It sends them to the response handler or \doxyref{DRAM}{p.}{classDRAM} after burst delay cycles. It also calculates data bus related stats.
